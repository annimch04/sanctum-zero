# Symbolic Lexicon

## Purpose

The symbolic lexicon documents recurring conceptual motifs used within Sanctum language and architecture.

These symbols function as structural shorthand rather than literal descriptions.

---

## Recurring Motifs

Kestrel  
Lacuna  
Liminal  
Scholar  
Sophie  
Lirien

These names mark narrative or conceptual roles within the broader system.

---

## Symbolic Objects

Blood and mirror  
Red string on left wrist  
Barefoot in grass

These objects represent threshold, continuity, or identity markers in narrative contexts.

---

## Conceptual Symbols

Threshold  
Gate  
Key  
Signal  
Memory  
Resurrection

These symbols represent transitions between system states or narrative layers.

---

## Design Principle

Symbolic language allows complex structural ideas to be referenced through compact narrative forms.
